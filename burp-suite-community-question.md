# Burp Suite Community Edition - Student Assessment Questions with Demo

## 1. Proxy

### Knowledge Questions
1. What is the purpose of the Proxy tool?
2. What does "intercepting a request" mean?
3. What is the difference between an HTTP request and response?
4. Why is Proxy considered the core feature of Burp Suite?
5. Why is a Burp certificate needed for HTTPS traffic?

### Practical Questions
1. How do you turn interception on and off?
2. How would you identify login credentials in a request?
3. Which part of a request contains cookies?
4. How would you modify a parameter before forwarding it?

## 2. Target (Site Map)

### Knowledge Questions
1. What is the purpose of the Site Map?
2. Why should testers map an application before testing it?
3. What information can be discovered from the Site Map?
4. What is the difference between a page and an API endpoint?

3. Identify API endpoints.

## 3. HTTP History

### Knowledge Questions
1. What is stored in HTTP History?

3. What information can be extracted from a request?

### Practical Questions

2. Find a login request.

## 4. Repeater

### Knowledge Questions
1. What is Repeater used for?
2. Why would a tester resend the same request multiple times?
3. What should be compared between responses?

### Practical Questions
1. Send a request to Repeater.
2. Change a parameter and resend it.
3. Compare the original response with the modified response.
4. Identify changes in status codes and content.
